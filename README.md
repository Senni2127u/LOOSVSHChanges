# LOOS VSH Changes
These are changes that have been done on the Vs Saxton Hale (VScript) LOOS Community server.

-----------------------------------------------------------------------------
Credits:

Bradasparky, Horiuchi, Dice, Senni - Coding and Technical Assistance.

Pika - Top Damage Scoreboard script maker.

Lizard of Oz - VSH main developer and some existing code.

-----------------------------------------------------------------------------

CURRENT CHANGES:

Saxton Hale - 

* Extender has been fixed.
* Saxton Hale has innate immunity to Pyro's Airblast stun mechanic.
* Saxton Punch voicelines happen 3 seconds prior to it occurring, and a Crit sound effect will play when active.

Scout - 

* Backscatter accuracy penalty is removed.
* Crit-a-Cola and Bonk! Atomic Punch have reduced cooldowns by 35%
* Crit-a-Cola's Marked for Death attribute was removed.

Soldier - 

* Gunboats cancel falling damage.

Pyro -

* Pyro now has 200 base HP, overheal remains the same.
* Axtinguisher gives half HP back upon successful hit (no overheal)
* Gas Passer Explodes on Ignite, but requires 800 damage to charge instead of 750.
* Gas Passer's Holster/Deploy speed was reduced by 15%.
* Manmelter gains stored criticals based on primary weapon damage.
* Airblast only works if done directly above or below Saxton Hale.

Demoman -

* Ullapool Caber now recharges after 15 seconds.

Heavy -

None.

Engineer -

* Teleporters can now be used in both directions.

Medic -

* Medic starts with 100% Ubercharge on all Medi Guns.

Sniper -

None

Spy -

* Cloak and Dagger mirrors stock Invis Watch statistics.
* Base Movement Speed is increased by 20%.
